# Work Report → Project Management Revenue 실적 자동 합산 공식

위치: **Project Management > View > Sales/Cost Performance Update Modal (실적 입력 모달)**

## 목표

기존 Pre-Sales 단계에서 입력한 "내부인력 매입 계획"이 그대로 흘러왔던 것처럼, **실적 단계에서는 Work Report에 기록된 실제 공수(M/D)를 등급별(특급/고급/중급/초급)로 집계하여 매입 실적으로 자동 반영**한다.

---

## 핵심 흐름

```
Work Report 업무내용 행 (사원별, 프로젝트별, 일자별)
    ↓
사원아이디(userId) → 직원마스터 → 직책 → 등급(특급/고급/중급/초급) 매핑
    ↓
업무구분=프로젝트 AND 프로젝트명=현재프로젝트인 행만 필터
    ↓
소요시간(분) ÷ 480 으로 M/D 환산
    ↓
등급별 × 월별로 합산
    ↓
M/D × 등급별 단가 = 월별 매입 실적 금액
    ↓
Project Management Revenue 모달의 [매입 > 내부인력 > 등급] 실적 행에 자동 입력
```

---

## 등급별·월별 실적 합산 공식

대상 모달에서 행 식별 조건:
- `data-row-type = "실적"`
- 구분 = 매입
- 세부구분 = 내부인력
- 등급(품명/규격) = 특급 / 고급 / 중급 / 초급

### 공식 1: 등급별 월별 M/D 합계

```
실적_MD[등급, 월] = SUM(
    Work Report 업무내용 행들 중
    [업무구분 = "프로젝트"]
    AND [프로젝트명 = 현재프로젝트]
    AND [작성자 직책의 매핑등급 = 등급]
    AND [업무일지 기준일의 월 = 월]
    의 소요시간(분) ÷ 480
)
```

### 공식 2: 등급별 월별 실적 금액

```
실적_금액[등급, 월] = 실적_MD[등급, 월] × 등급별_단가[등급]

등급별_단가:
  특급 = 10,000,000
  고급 =  9,000,000
  중급 =  7,000,000
  초급 =  5,000,000
```

### 공식 3: 등급별 총액 (실적 행의 총액 셀)

```
실적_총액[등급] = SUM(실적_금액[등급, 모든월])
            = 실적_MD_총합[등급] × 등급별_단가[등급]
```

### 공식 4: 등급별 총 수량(M/D) (실적 행의 수량/공수 셀)

```
실적_수량[등급] = SUM(실적_MD[등급, 모든월])
```

### 공식 5: 모달 하단 매입 합계

```
모달_하단_매입계 = SUM(모든 매입 행의 총액)
              = (기존 매입 행들의 총액)
                + 실적_총액[특급] + 실적_총액[고급] + 실적_총액[중급] + 실적_총액[초급]
```

---

## 직책 → 등급 매핑

| 직책 | 등급 |
|------|------|
| 이사, 상무, 전무, 부사장, 사장 | 특급 |
| 부장 | 고급 |
| 차장, 과장 | 중급 |
| 대리, 주임, 사원 | 초급 |

---

## 자동화 트리거 시점

다음 시점에 등급별 실적이 재계산되어 모달에 반영되어야 한다:

1. Work Report 신규 저장 시
2. Work Report 수정 저장 시
3. Work Report 삭제 시
4. 직원의 직책 변경(직원마스터 갱신) 시
5. Project Management View 진입 시 (실적 모달 열기 전 최신화)

---

## 데이터 식별자 매핑 (개발 참고)

| 항목 | Work Report 측 | Revenue 모달 측 |
|------|----------------|----------------|
| 프로젝트 식별 | 업무내용 행의 프로젝트명 셀(드롭다운 선택값) | 모달 상단 프로젝트ID |
| 월 식별 | 업무일지 기준일(seqDt)의 YYYYMM | 모달 컬럼의 YYYYMM |
| 작성자 | 사원아이디(userId) | (직접 매핑되지 않음, 등급으로만 합산) |
| 등급 | 사원아이디 → 직원마스터.직책 → 등급 매핑 | 행의 품명/규격 셀 |
| 공수 | 소요시간(분) ÷ 480 | 행의 수량/공수 셀 |
